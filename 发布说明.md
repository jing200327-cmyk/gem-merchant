# 宝石商人：给玩家的启动说明

## 双击启动（在线优先）

双击 `启动游戏.bat` 后，脚本会先检查并打开在线版本：

<https://jing200327-cmyk.github.io/-/>

在线版本可访问时，玩家无需安装任何软件；浏览器会直接打开游戏。

## 在线不可用时的本地回退

如果网页尚未部署、网络不可用或 GitHub Pages 暂时不可访问，启动器会自动改为启动本地游戏：

1. 安装 Node.js 20 或更高版本：<https://nodejs.org/>。
2. 双击 `启动游戏.bat`。
3. 浏览器将打开 <http://127.0.0.1:4173>。

本地游戏运行时，请保持弹出的命令行窗口处于打开状态；关闭它即可停止本地服务器。

## 当前在线地址的部署条件

在线地址需要项目成功推送到 GitHub，并由仓库的 GitHub Actions 完成 Pages 部署后才会生效。当前发布包始终可离线运行；在线地址暂不可用时，启动器会自动使用上述本地回退方案。

## 常见问题

- 浏览器没有自动打开：手动访问 <http://127.0.0.1:4173>。
- 端口已被占用：关闭其他正在运行的宝石商人窗口后重试。
- 手机游玩：待在线版本部署成功后，使用在线地址打开即可。
