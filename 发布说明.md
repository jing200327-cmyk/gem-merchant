# 宝石商人：给玩家的启动说明

## 最快的离线玩法

1. 解压 `宝石商人-发布包-v0.3.0.zip`。
2. 确认电脑安装了 Node.js 20 或更高版本：<https://nodejs.org/>。
3. 双击 `启动游戏.bat`。
4. 浏览器会自动打开 `http://127.0.0.1:4173`，选择玩家人数和 AI 后开始游戏。

游戏运行时请不要关闭弹出的命令行窗口。结束游戏后，关闭该窗口即可停止本地服务器。

## 没有 Node.js 怎么办？

当前版本使用原生 ES Modules；直接双击 `index.html` 可能被浏览器的本地文件安全策略拦截。请安装 Node.js 后使用启动脚本，或等待项目部署到网页版本。

## 在线游玩

仓库推送到 GitHub 后，维护者可在仓库设置中启用 **GitHub Pages**。部署完成后，玩家只需打开公开链接，无需下载或安装 Node.js。

## 常见问题

- 浏览器没有自动打开：手动访问 <http://127.0.0.1:4173>。
- 端口已被占用：关闭其他正在运行的宝石商人窗口后重试。
- 想在手机上玩：使用 GitHub Pages 等在线部署链接；本地 `127.0.0.1` 只指向当前电脑。
