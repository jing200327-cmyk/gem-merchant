# 宝石商人：给玩家的启动说明

## 直接在线启动

双击 `启动游戏.bat`，浏览器会立即打开在线游戏：

<https://jing200327-cmyk.github.io/-/>

无需复制链接、无需安装 Node.js，也不需要打开命令行窗口。

## 在线地址的部署条件

该地址需要项目成功推送到 GitHub，并由仓库的 GitHub Actions 完成 Pages 部署后才会显示游戏。如果浏览器显示 404 或页面不可用，说明在线版本尚未部署完成，而不是启动器需要复制粘贴地址。

## 本地运行（仅开发者）

若需要离线开发或调试，请安装 Node.js 20 或更高版本：<https://nodejs.org/>，然后在项目目录执行：

```powershell
node server.mjs
```

再访问 <http://127.0.0.1:4173>。
